# Bankist Project

This page emulates a Bank website application to prove the use of arrays and it's methods, and also the use of Numbers, Timers and Dates.

## Features

- Log in user

- Sort all movements

- Transfer money to other accounts

- Request loans

- Log out after 5 minutes of innactivity

- Close account

- See the money in the format of your country

- See the date in real time
